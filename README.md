### 情景：网络带宽分配
示例：网络服务商给企业客户A，B，C分配网络带宽，总带宽1000Mbps。A已占用300Mbps，还需200Mbps，B已占用400Mbps，还需150Mbps，C已占用0Mbps，还需300Mbps，给出合理分配策略（即找出安全序列）
